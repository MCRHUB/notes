核心指标列表

一、营销投入与规划类指标
营销总预算：周期内营销活动总投入金额（含渠道投放、活动落地、内容制作、人员相关等全链路营销支出）
单渠道预算占比：单渠道投入金额 / 营销总预算 × 100%
单线索目标成本：营销总预算 / 周期内目标线索总量单 MQL
目标成本：营销总预算 / 周期内目标 MQL 总量

二、渠道获客与流量类指标
曝光量：周期内营销内容、活动、广告被用户触达查看的总次数
点击量：周期内用户点击营销内容、落地页、官网链接的总次数
访问量：周期内用户访问营销落地页、官网的总人次
点击率（CTR）：点击量 / 曝光量 × 100%
渠道访问占比：单渠道访问量 / 总访问量 × 100%
落地页平均访问时长：周期内用户单次访问落地页的平均停留时长
落地页跳出率：仅访问 1 个页面就离开的用户数 / 落地页总访问用户数 × 100%
新增私域用户数：周期内通过营销动作新增的企业微信、公众号、社群等私域池用户总数

三、线索生成与归集类指标
总线索获取量：周期内全渠道归集、去重后的有效线索总数
单渠道线索获取量：周期内单一渠道带来的有效线索总数线索转化率：有效线索数 / 落地页总访问用户数 × 100%
单线索获取成本（CPL）：单渠道投入金额 / 该渠道有效线索获取量
线索去重率：去重后有效线索数 / 原始归集线索总数 × 100%
线索档案完整率：核心必填字段完整的线索数 / 总有效线索数 × 100%

四、线索培育与互动类指标（营销自动化 MA 核心）
线索培育覆盖率：进入自动化培育旅程的线索数 / 总待培育线索数 × 100%
内容触达量：周期内培育内容（邮件、短信、企微消息等）的总触达次数
内容触达用户数：周期内成功收到培育内容的独立用户总数
内容打开率：内容打开用户数 / 内容成功触达用户数 × 100%
内容点击率：内容内链接点击用户数 / 内容打开用户数 × 100%
线索互动率：周期内产生有效互动行为（打开、点击、报名、咨询等）的线索数 / 总培育线索数 × 100%
线索活跃度提升率：培育后活跃度完成升级的线索数 / 总培育线索数 × 100%
培育旅程完成率：完成全流程培育旅程的线索数 / 进入培育旅程的总线索数 × 100%
线索退订率：申请退订培育内容的线索数 / 总培育触达线索数 × 100%
培育线索流失率：周期内无任何有效互动的沉睡线索数 / 总培育线索数 × 100%

五、线索质量与合格认证类指标（MTL 核心里程碑）
MQL 生成总量：周期内通过市场部认证、符合目标客户画像、具备明确需求意向的营销合格线索总数
线索转 MQL 转化率：MQL 生成总量 / 总有效线索数 × 100%
单 MQL 获取成本：营销总投入金额 / MQL 生成总量 × 100%
MQL 移交总量：周期内市场部向销售部正式移交的 MQL 总数
MQL 转 SQL 转化率：通过销售部复核认证的销售合格线索 SQL 总数 / 总移交 MQL 数量 × 100%
SQL 生成总量：周期内通过销售部认证、具备明确商机潜力、可进入 LTC 销售流程的销售合格线索总数
线索退回率：销售部退回的无效 MQL 数量 / 总移交 MQL 数量 × 100%
线索平均流转时长：线索从生成到正式移交销售部的平均时长
MQL 平均跟进响应时长：销售部对移交 MQL 完成首次跟进的平均时长

六、线索转化与价值类指标（衔接 LTC/MCR）
SQL 转商机转化率：成功转为正式商机的 SQL 数量 / 总 SQL 生成总量 × 100%
线索成单总量：周期内由市场线索最终完成成交的订单总数
线索成单率：线索成交订单数 / 总有效线索数 × 100%
MQL 成单率：MQL 最终成交订单数 / 总 MQL 生成总量 × 100%
SQL 成单率：SQL 最终成交订单数 / 总 SQL 生成总量 × 100%
营销贡献营收：周期内由市场线索带来的成交订单总金额
营销线索平均客单价：营销线索成交总金额 / 营销线索成交订单总数
线索平均回款周期：线索从生成到最终完成全款回款的平均时长

七、营销 ROI 与效率类指标
营销投资回报率（ROI）：（营销贡献营收 - 营销总投入） / 营销总投入 × 100%
单渠道 ROI：（单渠道贡献营收 - 单渠道投入金额） / 单渠道投入金额 × 100%
营销费用率：营销总投入金额 / 营销贡献营收 × 100%
人均 MQL 产出：周期内总 MQL 生成量 / 营销团队在岗人数
人均营销贡献营收：周期内营销贡献营收总额 / 营销团队在岗人数

八、MTL 流程健康度监控指标
线索响应及时率：销售在规定时效内完成首次响应的 MQL 数量 / 总移交 MQL 数量 × 100%
线索跟进覆盖率：销售完成首次有效跟进的 MQL 数量 / 总移交 MQL 数量 × 100%
跨部门协同异议率：市场与销售产生质量异议的线索数量 / 总移交 MQL 数量 × 100%
培育流程断点率：培育旅程中出现中断、异常的线索数 / 总进入培育旅程的线索数 × 100%
数据完整度达标率：全链路核心指标数据完整的统计周期数 / 总统计周期数 × 100%
线索老化率：超过规定时长未跟进 / 未培育的线索数 / 总有效线索数 × 100%
